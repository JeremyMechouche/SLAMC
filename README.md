# SLAMC

This model represent the approach proposed in "Towards higher-level description of SLA-aware reconfiguration strategies based on state-machine". Its an SLA-aware reconfiguration  strategy  model  using  the  eclipse modeling  framework (EMF). It define the structure of an SLA, its objectives and integrates reconfiguration strategies as state-machine.
